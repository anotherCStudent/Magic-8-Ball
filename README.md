# Magic 8 Ball

A simple Magic 8 Ball game made with Python and Pygame, featuring custom graphics created by me.

## What it does

* Lets you type a question into the text box
* Click **Ask Question** to get a random Magic 8 Ball style answer
* Displays the answer inside the triangle on the ball
* Click **Quit** to exit

## Built With

* Python
* Pygame

## Run from Source

### Requirements

* Python 3
* Pygame

### Install

```bash
pip install pygame
```

### Run

```bash
python magic_8_ball.py
```

## Controls

* Type your question in the input box
* Click **Ask Question** to generate an answer
* Click **Quit** to close the game
